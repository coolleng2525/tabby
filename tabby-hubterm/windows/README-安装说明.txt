Tabby HubTerm Windows 安装说明

1. 完全退出 Tabby。
2. 双击 install.cmd。
3. 重新启动 Tabby。
4. 打开“设置 → HubTerm”。
5. 启用 HubTerm，并填写 Center URL，例如：
   ws://192.168.1.55:8097/api/ws/agent
6. Token 留空；首次打开终端标签时会自动注册并保存。

插件安装目录：
%APPDATA%\tabby\plugins\node_modules\tabby-hubterm

注意：coolleng2525/tabby 的自定义构建已内置该插件，不要重复安装。
